# Task starter — landing page hero

Use with `system-prompt.md` loaded.

Structure:
- **Headline** (≤8 words, display-font energy, may end on the brand period)
- **Subheadline** (1–2 sentences, includes a real number or data claim)
- **Primary CTA** (uppercase mono label, 2–4 words)
- **Secondary CTA** (optional)

Current production hero for reference:
- H1: "Stop guessing. KNOW what it's worth."
- Sub: "RentWise uses billions of data points to evaluate properties in Bangalore."
- CTA: "BROWSE COLLECTION" / "LANDLORD ACCESS"

Rules: verdict first, no "premium/luxury", numbers beat adjectives, CTA is an action not an invitation ("SHOW THE COMPS" not "Learn more").
