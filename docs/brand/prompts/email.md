# Task starter — email

Use with `system-prompt.md` loaded.

Rules:
- Subject ≤ 45 chars, contains a number or a verdict where possible.
  - Good: "Your HSR shortlist: 2 over market" / "Indiranagar moved 4% while you slept"
  - Bad: "Exciting updates from RentWise!"
- Body: max 3 short paragraphs. First line = the takeaway.
- One CTA button, uppercase mono label.
- Sign-off: "— RentWise" (em dash, no "Team", no "Cheers").
- Transactional emails state facts; marketing emails state market facts. Same voice either way.
