# Task starter — error / empty / loading states

Use with `system-prompt.md` loaded.

Rules:
- State what happened, what we're doing, what the user can do. ≤ 2 sentences.
- Warm but not cute. Specific but not technical. Never blame the user.
- No exclamation marks, no "Oops!", no sad-face emoji.

Reference set:
- Error: "Market data is syncing. Verdicts return in about a minute."
- Empty: "Zero matches. Loosen one filter — rent ceiling moves results most."
- Loading: "Pricing 1,432 comps…"
- 404: "This page does not exist. The listing may have been removed, or the address was mistyped."

Task: [DESCRIBE THE STATE — where in the app, what failed, what recovery exists]
